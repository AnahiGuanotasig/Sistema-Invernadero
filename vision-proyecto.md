# Sistema de gestión de invernadero para hogares


## OBJETIVO

Desarrollar un sistema de gestión web, donde es pueda hacer un regitro de las plantas
su desarrollo, observaciones y llevar un mejor control.

## Problema

Actualmente no se cuenta con un sistema de control, la evaluación de plantas se realiza
de manera individual sin llevar un registro que nos de registros de estado de cada planta.

## Usuarios

    Admnistrador
    Trabajador

## Funciones iniciales

    - Registro inicial de plantas(Especie, tamaño, estado de follaje, tipo, intervalos de riego, con notificaciones)
    - Registrar crecimiento (observaciones)
    - Registrar riego
    - Ver el historial
    - Administrar plantas (CRUD)
## Tecnologias
Backend
    Node.js + Express
    PostgreSQL
Frontend
    React
Arquitectura
    MVC(Modelo vista controlador)
